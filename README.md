# SEDIF Home Assistant Add-on

Dépôt Home Assistant Add-on pour récupérer la consommation d'eau SEDIF.

## Installation via Home Assistant

1. Ouvrir **Paramètres > Modules complémentaires > Boutique**.
2. Cliquer sur le menu ⋮ puis **Dépôts**.
3. Ajouter l'URL du dépôt :
   ```
   https://github.com/lfpoulain/ha-sedif
   ```
4. Installer l'addon **SEDIF Water Consumption**.

## Contenu du dépôt

- `repository.json` : métadonnées du dépôt d'addons Home Assistant.
- `ha-sedif-addon/` : dossier de l'addon.

## Documentation

La documentation détaillée et la configuration de l'addon se trouvent dans :
